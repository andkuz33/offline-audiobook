    Синхронный переводчик

    Приложение слушает вашу речь, распознаёт её, переводит на выбранный язык и при желании озвучивает перевод — в реальном времени, прямо в браузере.

    • Распознавание речи — Web Speech API браузера.
    • Перевод — бесплатный сервис MyMemory (нужен интернет).
    • Озвучивание — встроенный синтез речи браузера.

    Лучше всего работает в Google Chrome или Microsoft Edge на компьютере и Android. Для распознавания речи требуется доступ к микрофону и подключение к интернету.

    Никакие записи не сохраняются на сервере. Текст для перевода отправляется в сервис MyMemory.
