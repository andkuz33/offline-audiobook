# Синхронный переводчик

Синхронный (почти реального времени) перевод речи на переговорах.
Слушает окружающих на **английском/китайском** и озвучивает перевод **по-русски** вам в наушник; ваша русская речь переводится на **EN/ZH** для собеседника.

В репозитории две реализации:

## 📱 `ios/` — нативное приложение iOS (основное направление)

Прототип на **SwiftUI + Gemini Live API** (потоковое аудио в обе стороны):

- **«Слушаю окружающих»** — микрофон телефона, авто-определение EN/ZH → русский голос в AirPods.
- **«Говорю»** (push-to-talk) — русская речь → EN или ZH (тумблер) → из динамика телефона.

Сборка и подробности — в [`ios/README.md`](ios/README.md). Нужен Mac + Xcode + ключ Gemini, запуск на реальном iPhone.

## 🌐 `web/` — браузерный PWA (ранний прототип)

Веб-версия на чистом HTML/CSS/JS: распознавание речи браузера → перевод (MyMemory) → озвучивание. 14 языков, работает без установки. Подробности — в [`web/README.md`](web/README.md).

---

## Статус

Прототипы. Ключевая часть iOS-конвейера (захват → стрим в Gemini Live → озвучка) собрана, но требует проверки на устройстве. Дальнейшие шаги — авто-VAD вместо push-to-talk, субтитры, автоопределение языка собеседника.
