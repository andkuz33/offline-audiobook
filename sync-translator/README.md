# Синхронный переводчик

Веб-приложение для синхронного перевода речи прямо в браузере. Работает без установки и без серверной части.

**Поток работы:** распознавание речи → перевод → озвучивание, в реальном времени.

## Возможности

- 🎙️ Распознавание речи через Web Speech API браузера.
- 🌐 Перевод на 14 языков через бесплатный сервис [MyMemory](https://mymemory.translated.net/) (без ключа).
- 🔊 Озвучивание перевода встроенным синтезом речи.
- ♻️ Непрерывный режим — для последовательного устного перевода.
- 🔁 Кнопка обмена языками местами.
- 📜 История переведённых фраз с повторным озвучиванием.
- 📱 PWA: устанавливается на телефон/компьютер, интерфейс открывается офлайн.

## Как запустить

Это статический сайт — достаточно открыть `index.html`. Но для микрофона и Service Worker нужен контекст `https://` или `localhost`:

```bash
# из папки sync-translator
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Или опубликуйте папку через GitHub Pages.

## Требования

- Браузер с поддержкой Web Speech API — **Google Chrome** или **Microsoft Edge** (десктоп и Android).
- Доступ к микрофону.
- Подключение к интернету для распознавания и перевода (офлайн открывается только интерфейс).

## Приватность

Записи речи не сохраняются на сервере. Распознанный текст отправляется в сервис MyMemory только для получения перевода.

## Технологии

Чистый HTML/CSS/JavaScript, без сборки и зависимостей. Web Speech API (`SpeechRecognition`, `SpeechSynthesis`), Service Worker, Web App Manifest.
