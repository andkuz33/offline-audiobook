# Offline AudioBook Web

Веб-приложение для прослушивания MP3-аудиокниг офлайн на Windows и iPhone.

## Как пользоваться на Windows

1. Откройте папку `OfflineAudioBookWindows`.
2. Дважды нажмите `start.bat`.
3. Откроется браузер по адресу `http://127.0.0.1:8765`.
4. Нажмите «Добавить книгу».
5. Выберите MP3-файл.

## Как пользоваться на iPhone через Chrome

1. Откройте опубликованный сайт в Chrome на iPhone.
2. Нажмите «Добавить книгу».
3. Выберите MP3 из приложения «Файлы».
4. Если хотите иконку на главном экране: нажмите «Поделиться» и попробуйте «Добавить на главный экран».
5. Если Chrome не показывает этот пункт, откройте ту же ссылку в Safari и добавьте на главный экран оттуда.

На iPhone все браузеры используют системный движок Apple, поэтому Chrome и Safari имеют похожие ограничения.

## Как выложить на GitHub Pages

1. Создайте новый репозиторий на GitHub, например `offline-audiobook`.
2. Загрузите в него все файлы из папки `OfflineAudioBookWindows`.
3. Откройте настройки репозитория.
4. Перейдите в `Pages`.
5. В `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Нажмите `Save`.
7. Через минуту GitHub даст ссылку вида `https://ваш-логин.github.io/offline-audiobook/`.
8. Откройте эту ссылку на iPhone.

## Что сохраняется

- импортированные MP3;
- позиция прослушивания;
- скорость;
- прогресс;
- библиотека.

Данные хранятся внутри браузера на конкретном устройстве. Если очистить данные сайта или браузера, книги удалятся из локального хранилища.

## Ограничения iPhone

- Фоновое воспроизведение обычно работает, но зависит от iOS и браузера.
- Управление с экрана блокировки может работать не так стабильно, как в настоящем App Store приложении.
- iOS может очистить данные сайта при нехватке памяти.
- Для выбора файлов из Google Drive, Dropbox, Яндекс.Диска или iCloud Drive эти сервисы должны быть подключены в приложении «Файлы».

## Возможности

- Импорт `.mp3`.
- Офлайн-хранение в IndexedDB.
- Продолжение с последней позиции.
- Автосохранение каждые 5 секунд.
- Скорости `1x`, `1.25x`, `1.5x`, `2x`, `2.5x`.
- Кнопки `−15` и `+30`.
- Возобновление с кнопки на наушниках после прерывания звонком или другим приложением (плеер переустанавливает источник и возвращается на сохранённую позицию).
- Автоматическое открытие последней книги при запуске (можно сразу нажать кнопку на наушниках).
- Закладки с текстовыми комментариями: добавление на текущей позиции, переход, редактирование комментария и удаление. Комментарий вводится в многострочном модальном окне, а количество закладок показывается на карточке книги в библиотеке.
- История прослушивания: приложение фиксирует, когда вы начали слушать книгу, и длительность каждого сеанса (перерыв дольше минуты считается новым сеансом).
- Отчёт о прослушивании по кнопке «Отчёт» в плеере: когда начато, последнее прослушивание, всего прослушано, прогресс, число сеансов и история. Кнопка «Сохранить картинку» создаёт PNG-отчёт (через «Поделиться» на iPhone или скачиванием) — удобно для скриншота. Кнопка «Экспорт в текст» сохраняет историю в `.txt`.
- Минималистичный русский интерфейс.
- Кэширование сайта для офлайн-открытия после первого запуска.
