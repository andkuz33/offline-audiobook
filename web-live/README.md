# Синхронный переводчик — веб-версия (Gemini Live)

Работает прямо в браузере iPhone (Safari) — **без Mac, без App Store**. Микрофон телефона слышит речь, Gemini Live переводит, перевод звучит в наушнике/динамике.

- **Слушать окружающих:** EN/ZH → русский голос вам в AirPods (язык определяется автоматически).
- **Говорить (зажать кнопку):** русская речь → English или 中文 (тумблер) → из динамика телефона собеседнику.
- Субтитры исходной речи и перевода на экране.

## Как запустить

Это статический сайт, но микрофону нужен `https://` (или `localhost`).

**Вариант 1 — GitHub Pages (рекомендуется для iPhone):**
1. Включите Pages для репозитория (Settings → Pages → ветка/каталог с этими файлами).
2. Откройте полученный `https://…`-адрес в Safari на iPhone.

**Вариант 2 — локально:**
```bash
cd web-live
python3 -m http.server 8000
# открыть http://localhost:8000 (микрофон работает на localhost)
```

## Ключ Gemini

1. Получите ключ в [Google AI Studio](https://aistudio.google.com/apikey).
2. В приложении нажмите **«⚙︎ Ключ»**, вставьте его и сохраните. Ключ хранится только в вашем браузере (localStorage).

## Как пользоваться на переговорах

1. Вставьте AirPods, откройте сайт в Safari, добавьте на экран «Домой» (Поделиться → «На экран Домой») — будет как приложение.
2. Нажмите **«Слушать окружающих»** — речь на EN/ZH зазвучит по-русски.
3. Чтобы ответить — выберите EN/ZH, **зажмите** кнопку микрофона и говорите по-русски; перевод выйдет из динамика.

## Важные ограничения веб-версии

- **Только пока экран включён и вкладка открыта** — Safari усыпляет фоновые вкладки. Для «телефон на столе во время разговора» это нормально, для постоянного фонового перевода нужна нативная версия.
- **Маршрут микрофона.** С подключёнными AirPods Safari может брать звук с микрофона AirPods, а не телефона — контроля меньше, чем в нативном приложении. Если ловит плохо, попробуйте без AirPods на вход (динамик) или один наушник.
- **Ключ виден в браузере.** Для личного пользования ок; для публичного сервиса вынесите вызовы Gemini на свой сервер-прокси.
- **Задержка ~1–3 с** — «почти синхронно», модели нужно услышать фрагмент фразы.
- **Модель** `gemini-2.0-flash-live-001`. Поменять можно в `gemini-live.js`. Модели «Gemini 3.5 Flash» не существует.

## Файлы

| Файл | Назначение |
|------|-----------|
| `index.html`, `styles.css` | интерфейс |
| `app.js` | режимы, оркестрация, ключ |
| `gemini-live.js` | WebSocket-клиент Gemini Live API |
| `audio.js` | захват микрофона (PCM16 16кГц) и воспроизведение (24кГц) |
| `pcm-worklet.js` | AudioWorklet для захвата кадров микрофона |
| `service-worker.js`, `manifest.webmanifest` | установка как PWA |
