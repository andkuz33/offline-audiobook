# Синхронный переводчик

Синхронный (почти реального времени) перевод речи на переговорах через **Gemini Live API**. Слушает окружающих на **английском/китайском** и озвучивает перевод **по-русски** вам в наушник; ваша русская речь переводится на **EN/ZH** для собеседника.

Основная версия — **веб-приложение** (файлы в корне репозитория): работает прямо в **Safari на iPhone**, без Mac и App Store. Микрофон телефона → Gemini Live → перевод в AirPods. Есть субтитры, устанавливается на экран «Домой» как PWA.

- **«Слушать окружающих»** — авто-определение EN/ZH → русский голос в наушник.
- **«Говорить»** (зажать кнопку) — русская речь → English или 中文 (тумблер) → из динамика телефона.

## Быстрый старт

1. Включите **GitHub Pages**: Settings → Pages → Source: ветка `main`, папка `/ (root)`.
2. Откройте полученный `https://andkuz33.github.io/sync-translator/` в **Safari** на iPhone.
3. Нажмите **«⚙︎ Ключ»**, вставьте ключ из [Google AI Studio](https://aistudio.google.com/apikey) (хранится только в браузере).
4. Вставьте AirPods, нажмите **«Слушать окружающих»**. Чтобы ответить — выберите EN/ZH, зажмите микрофон и говорите по-русски.

## Ограничения веб-версии

- Работает, **пока экран включён и вкладка открыта** (Safari усыпляет фон) — для «телефон на столе во время разговора» нормально.
- С AirPods на входе Safari может брать их микрофон, а не телефона — контроля меньше, чем в нативном.
- Ключ виден в браузере (ок для личного пользования; для продакшена нужен серверный прокси).
- Задержка ~1–3 с — «почти синхронно».
- Модель `gemini-2.0-flash-live-001` (меняется в `gemini-live.js`).

## Статус

Прототип, пригоден к использованию на iPhone сразу. Дальше: авто-VAD вместо push-to-talk, надёжное переподключение, серверный прокси для ключа.
